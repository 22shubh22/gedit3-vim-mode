gedit3-vim-mode
================

This plugin enhances gedit3 with basic Vim-style keyboard navigation. To install, 
just copy vim-mode.py and vim-mode.plugin to ~/.local/share/gedit/plugins/ . This
requires PyGObject.

Features
--------
* normal mode cursor navigation
* repeating motions
